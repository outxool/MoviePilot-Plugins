# TMDB趋势订阅插件

自动订阅TMDB热门/趋势的电影、电视剧、动画等内容，支持评分过滤和分类配置。

## 功能特点

1. **多分类支持**：
   - 电影（趋势、流行、高分、正在上映、即将上映）
   - 电视剧（趋势、流行、高分、正在播出、今日播出）
   - 动画（趋势、热门）
   - 纪录片（趋势、热门）

2. **智能过滤**：
   - 自定义最低评分（vote_average）
   - 最少评价数过滤
   - 排除成人内容选项

3. **灵活配置**：
   - 每个分类单独启用/禁用
   - 每个分类类型单独配置
   - 自定义最大订阅数量
   - 定时任务执行周期

4. **通知提醒**：
   - 添加新订阅时发送通知
   - 任务完成时发送统计通知

## 安装方法

### 方法一：通过GitHub仓库安装（推荐）

1. 在MoviePilot插件库中，点击"添加插件源"
2. 输入GitHub仓库地址：`https://github.com/yourusername/moviepilot-tmdb-trending`
3. 在插件市场中找到"TMDB趋势订阅"并安装

### 方法二：手动安装

1. 下载插件文件
2. 将插件文件夹放到MoviePilot的插件目录：`/config/plugins/`
3. 重启MoviePilot
4. 在插件市场中启用插件

## 配置说明

### 必需配置

1. **TMDB API Key**：
   - 访问 https://www.themoviedb.org/settings/api 获取API Key
   - 在插件配置中填入API Key

2. **执行周期**：
   - 默认每天9点执行
   - 支持cron表达式（如：`0 9 * * *`）

### 可选配置

1. **评分过滤**：
   - 最低评分：默认6.0分
   - 最少评价数：默认100个

2. **分类配置**：
   - 每个分类可单独启用
   - 设置每个分类的最大订阅数量
   - 设置每个分类的最低评分

## 使用说明

1. **配置插件**：
   - 填写TMDB API Key
   - 根据需要启用分类和类型
   - 设置评分过滤条件

2. **立即执行**：
   - 在配置页面勾选"立即运行一次"
   - 或使用命令：`/tmdb_sync`

3. **查看历史**：
   - 在插件详情页面查看订阅历史
   - 点击标题可跳转到TMDB页面

## 注意事项

1. TMDB API有请求频率限制，请合理设置执行周期
2. 建议每天执行一次，避免频繁请求
3. 如果订阅数量过多，建议减少最大订阅数量
